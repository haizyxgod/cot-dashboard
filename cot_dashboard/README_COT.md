# 🚀 COT Dashboard - Инструкция по использованию

## 📊 Что это?

COT Dashboard - это локальный веб-сайт для отслеживания и анализа данных COT (Commitments of Traders) по трем инструментам:
- **XAU (Золото)** 🥇
- **EUR/USD** 💶
- **GBP/USD** 💷

## 🌐 Как открыть сайт?

Откройте браузер и перейдите по одному из адресов:
- **http://localhost:5000** (локальный доступ)
- **http://127.0.0.1:5000** (альтернативный локальный адрес)
- **http://192.168.43.83:5000** (доступ из локальной сети)

## ✨ Возможности

### 1. Просмотр текущих позиций
- Открытый интерес
- Позиции спекулянтов (Long/Short)
- Нетто-позиции
- Настроение рынка (Бычий/Медвежий)

### 2. Анализ изменений
- Изменения за неделю
- Автоматический анализ трендов
- Визуальные индикаторы (↑↓)

### 3. Автоматическое обновление
- **Каждое воскресенье в 10:00** данные обновляются автоматически
- Следующее обновление: **31 мая 2026 в 10:00**

## 🔄 Ручное обновление

Нажмите кнопку **"Обновить"** в правом верхнем углу сайта для принудительного обновления данных.

## 📊 Интерпретация данных

### Золото (XAU)
- **Спекулянты (Некоммерческие трейдеры)**: Крупные фонды и трейдеры
- **Положительная нетто-позиция** = Бычий настрой (ожидание роста)
- **Отрицательная нетто-позиция** = Медвежий настрой (ожидание падения)

### EUR/USD и GBP/USD
- **Leveraged Funds**: Спекулятивные фонды с кредитным плечом
- **Asset Managers**: Крупные управляющие компании
- **Положительная нетто-позиция** = Бычий настрой по валюте
- **Отрицательная нетто-позиция** = Медвежий настрой по валюте

## 🎯 Торговые сигналы

### Сильный бычий сигнал 🟢
- Резкое увеличение длинных позиций спекулянтов
- Рост открытого интереса
- Изменение нетто-позиции > +10,000 контрактов

### Сильный медвежий сигнал 🔴
- Резкое сокращение длинных позиций
- Увеличение коротких позиций
- Изменение нетто-позиции < -10,000 контрактов

### Нейтральный сигнал ⚪
- Незначительные изменения позиций
- Стабильный открытый интерес

## ⚙️ Настройка автообновления

По умолчанию данные обновляются **каждое воскресенье в 10:00**.

Чтобы изменить расписание, отредактируйте файл `app.py`:

```python
scheduler.add_job(
    func=update_data,
    trigger='cron',
    day_of_week='sun',  # 0=пн, 6=вс
    hour=10,            # Час (0-23)
    minute=0            # Минута (0-59)
)
```

## 🔧 Устранение неполадок

### Сервер не запускается
```powershell
# Проверьте, что все зависимости установлены
pip install -r requirements.txt
```

### Порт 5000 занят
Измените порт в `app.py`:
```python
app.run(debug=True, host='0.0.0.0', port=5001)  # Используйте другой порт
```

### Данные не загружаются
```powershell
# Проверьте подключение к интернету и запустите
python cot_fetcher.py
```

## 📞 Источник данных

Данные получаются из официального API **CFTC** (Commodity Futures Trading Commission):
- https://publicreporting.cftc.gov/

Отчеты публикуются каждую пятницу в 15:30 ET с данными по состоянию на вторник.
